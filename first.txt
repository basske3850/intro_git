管理しないファイルをgitの管理から外す
.gitignore ファイルに指定することで、ファイルをGitの管理から外すことができる
自動生成されるファイルやパスワードが記載されているファイル

#から始まる行はコメント
index.html

コミットしてしまったファイルを管理から外す方法
git rm コマンドで、コミットしてしまったファイルをGIｔの管理から削除できる。

git rm [ファイル名]
git rm -r[ディレクトリ名]

ファイルを残す場合
ファイルは残すけどgit管理から外す
git rm --cached [ファイル名]


git rm second.txt で除外したファイルをもとの戻す場合は、

git reset HEAD second.txt で戻せる
